## Getting Started

1. `git clone https://github.com/pointzi/HackerNews.git`
2. `cd HackerNews`
3. `pod install` (requires [CocoaPods](https://cocoapods.org))
4. Open `HackerNews.xcworkspace` in Xcode 8.0 or higher.
